Sobre o Site

Um demonstrador de tecnologias aplicadas ao desenvolvimento web e à gestão de TI.

Finalidade didática

Este site foi desenvolvido como um demonstrador de tecnologias para uso em aulas de graduação e pós-graduação em disciplinas de Tecnologia da Informação, Sistemas de Informação, Desenvolvimento Web, Governança de TI e Gestão de TI.

O tema da cultura gaúcha serve como contexto para demonstrar conceitos práticos de construção de páginas, organização de conteúdo, responsividade, interatividade, acessibilidade, publicação e evolução incremental de sistemas web.

Uso de inteligência artificial

A página e partes do site foram construídas com apoio de inteligência artificial, utilizando a ferramenta OpenCode como ambiente de colaboração, edição e automação de tarefas de desenvolvimento.

Objetivos pedagógicos

  • Demonstrar construção incremental de um site estático com HTML, CSS e JavaScript.
  • Discutir requisitos, escopo, backlog, versionamento e qualidade em projetos de TI.
  • Exercitar responsividade, acessibilidade, organização de conteúdo e navegação hipertextual.
  • Relacionar cultura digital, preservação de memória, dados, governança e uso responsável de inteligência artificial.
  • Permitir que estudantes proponham melhorias como busca, filtros, mapas, APIs, banco de dados e publicação automatizada.

Tecnologias utilizadas

HTML5

Utilizado para estruturar as páginas, cabeçalhos, navegação, seções de conteúdo, listas, links e rodapés.

CSS3

Utilizado para identidade visual, cores, tipografia, sombras, cards, botões, layouts em grade e adaptação visual.

JavaScript

Utilizado para menu responsivo, identificação automática da página ativa, botão de voltar ao topo e interações no dicionário.

Design responsivo

Utilizado para adaptar o site a telas de computadores, tablets e celulares por meio de media queries e grids flexíveis.

Semântica web

Utilização de elementos como header, nav, main, section, article e footer para melhorar organização e leitura do código.

Acessibilidade básica

Inclui foco visível, botões com atributos ARIA, navegação por teclado e uso de textos de link compreensíveis.

Google Fonts

Utilização de fontes externas para demonstrar integração com serviços web e controle tipográfico via CSS.

Imagens WebP e favicon

Uso de imagem de fundo em formato WebP e ícone do site para demonstrar recursos comuns de apresentação web.

Links externos e mailto

Uso de links para sites institucionais e link de contato por e-mail no rodapé.

OpenCode

Ferramenta utilizada para apoiar a criação, alteração e revisão dos arquivos do site com assistência de IA.
